ICTCLASWrapper by guoquan

Java wrapper for ICTCLAS from ICT.

Dependencies

ICTCLAS (http://ictclas.org/) --ICTCLASWrapper 0.0.1 support only ICTCLAS 2011.

Install

Prepare ICTCLAS 2011 and everything required by ICTCLAS 2011, and ICTCLASWrapper is just here ready for your java code.
